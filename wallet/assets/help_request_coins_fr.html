En utilisant cette boîte de dialogue, vous pouvez recevoir des ArtBytes (vous devez avoir installé ArtByte Wallet).

D'abord, saisissez le montant de ArtBytes à recevoir.

Ensuite, l'autre personne doit scanner le QR code généré. Ou bien, vous pouvez envoyer une URL ArtByte à une personne à distance avec le bouton "publier" dans la barre d'action. L'autre personne devra cliquer l'URL ArtByte.

Dasn les deux cas, l'autre personne se verra présenter une boîte de dialogue pour envoyer des ArtBytes avec tous les champs pré-remplis. Si elle accepte, elle n'a plus qu'à envoyer la transaction.
